# Learning Path

This file outlines a suggested learning path for the repository. Update with links to notes and weekly goals.

1. Big O and algorithmic complexity
2. Arrays and sliding-window patterns
3. Strings, two-pointers, and hashing
4. Linked lists, fast-slow pointers
5. Trees, BFS/DFS
6. Graphs and shortest-paths
7. Dynamic programming
